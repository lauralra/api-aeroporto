# api-aeroporto
Construção de uma API REST para gerenciar o cadastro de aeroportos espalhados pelo mundo


# Objetivo

Esse projeto tem o objetivo de consturir uma API REST para gerenciar o cadastro de aeroportos, converter unidades de pés para metros e realizar as operações CRUD.

# Tecnologias utilizadas

Foram utilizados o Visual Studio Code e o Spring-boot

## Iniciar o spring-boot da aplicação

./mvnw spring-boot:run

## Realizar testes unitários

.\mvnw.cmd clean test

## Realizar testes unitários e de integração

.\mvnw.cmd clean verify

## Realizar testes de integração

.\mvnw.cmd clean integration-test
